# Sample Images

This directory contains sample images generated by the hybrid image generator, demonstrating all theme and layout combinations.

## Generated Samples

The `generate-samples.js` script creates 9 images:

### Themes
- **chalkboard** - Educational, hand-drawn style with dark green background
- **watercolor** - Professional, soft watercolor style with cream background
- **tech** - Technical, isometric 3D style with dark gradient background

### Layouts
- **comparison** - Side-by-side comparison (e.g., pros vs cons)
- **evolution** - Horizontal progression flow (e.g., stages 1 → 2 → 3)
- **single** - Deep dive on one topic with multiple sections

## File Naming Convention

```
{theme}-{layout}.png
```

Examples:
- `chalkboard-comparison.png` - Chalkboard theme with comparison layout
- `watercolor-evolution.png` - Watercolor theme with evolution layout
- `tech-single.png` - Tech theme with single-topic layout

## Generation

To generate all samples:

```bash
cd automation/hybrid-image-generator
node generate-samples.js
```

To generate specific combinations:

```bash
# Only chalkboard theme
node generate-samples.js --theme chalkboard

# Only comparison layout
node generate-samples.js --layout comparison

# Specific theme + layout
node generate-samples.js --theme watercolor --layout evolution
```

## Requirements

- OpenAI API key in `.env` (for DALL-E background generation)
- OR pre-cached backgrounds in `cache/backgrounds/`

## Sample Content

Each layout uses realistic content data:

- **Comparison**: Data Mesh vs Data Warehouse features and challenges
- **Evolution**: Data storage evolution from traditional DB → warehouse → lakehouse
- **Single**: Vector Databases deep dive with use cases

## Output

Each generated image:
- Resolution: 2160×2160 pixels (1080×1080 @ 2× scale)
- Format: PNG
- Includes metadata logged to console (theme, layout, generation time, cache stats)
